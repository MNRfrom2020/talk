# **App Name**: PodLink

## Core Features:

- Podcast URL Import: Allow users to add podcast episodes by providing a direct link to an MP3 file from GitHub or Drive.
- Personalized Library: Create a personalized library to store and organize saved podcast episodes.
- Audio Playback Controls: Implement comprehensive audio playback controls (play, pause, skip, volume, progress bar).

## Style Guidelines:

- Primary color: Deep purple (#6A478F) for a modern and engaging feel.
- Background color: Dark gray (#282828) for a modern dark theme.
- Accent color: Teal (#4DB6AC) for highlights and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, will be used for a modern, objective, and neutral aesthetic suitable for both headlines and body text.
- Use modern, minimalist icons for playback controls and library navigation.
- Design a clean and intuitive layout inspired by Spotify, with clear separation of sections.
- Incorporate subtle transitions and animations to enhance user experience.
